# RGB_tissues_analysis-filter

Aplikacja do filtrowania i analizy obrazu z wykorzystaniem przestrzeni kolorów RGB. Umożliwia przetwarzanie zdję (formaty: PNG, JPG, TIFF) poprzez separację kanałów barwnych oraz obliczanie powierzchni obszarów o określonym kolorze. Narzędzie wspiera analizę map FTIR/Raman, pomagając w ilościowej ocenie struktur widocznych na obrazach.

Aplikacja została stworzona w programie MATLAB.
